# Carlos Castro Portfolio

One-page Next.js portfolio for AI fellowship and operator/job applications.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Resume PDF

Place the resume file at:

```text
public/Carlos_Castro_AI_Operator_Resume_2026.pdf
```

The Resume button is already wired to `/Carlos_Castro_AI_Operator_Resume_2026.pdf`.
